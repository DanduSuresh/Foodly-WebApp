# Foodly WebApp

A Java Web Application for online food ordering.

## Tech Stack
- Java, JSP, Servlets
- JDBC, MySQL
- HTML, CSS, JavaScript
- Apache Tomcat

## Features
- User registration and login
- Browse restaurants and menus
- Add items to cart
- Place orders

## How to Run
1. Import the project into Eclipse
2. Configure Apache Tomcat 9
3. Create MySQL database and update DB config in code
4. Run on server and open in browser: `http://localhost:8080/Foodly`
